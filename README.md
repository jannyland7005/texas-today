# 텍사스, 오늘 — 웹앱

이 폴더를 GitHub Pages 또는 Netlify에 올리면 휴대폰에서 https 주소로 접속해 사용할 수 있습니다.

## GitHub Pages에 올리는 방법

1. GitHub에서 새 repository를 만듭니다. 예: `texas-today`
2. 이 폴더 안의 파일들을 모두 업로드합니다.
   - `index.html`
   - `manifest.webmanifest`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
3. Repository의 Settings → Pages로 이동합니다.
4. Source를 `Deploy from a branch`로 선택합니다.
5. Branch를 `main` / `/root`로 선택하고 Save를 누릅니다.
6. 잠시 후 아래와 같은 주소가 생깁니다.

`https://사용자이름.github.io/texas-today/`

## 휴대폰에서 사용

1. 위 https 주소를 휴대폰 브라우저에서 엽니다.
2. 사진을 업로드하고 문구를 입력합니다.
3. “결과물 다른 앱으로 공유하기”를 눌러 카카오톡, 문자, 이메일 등으로 공유합니다.
4. 홈 화면에 추가하면 앱처럼 사용할 수 있습니다.
